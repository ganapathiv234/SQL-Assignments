# SQL-Assignments
This project explores an e-learning platform database using SQL for schema design, data insertion, and analytical queries. It includes a visual dashboard summarizing key insights from learner behavior, course performance, and revenue trends.
The infographic highlights:
- 📊 Revenue by Category → Technology dominates, followed by Business and Finance, while Arts lags.
- 👩‍🎓 Top Learners by Spending → Aaliya (India), Chen Wei (China), and Innaiya Fathima (USA) lead in total spending.
- 🏆 Top Courses by Quantity Sold → Python Fundamentals, Financial Accounting, and Business Communication are the most popular.
- 🔄 Multi-Category Learners → About 25% of learners purchased across multiple categories.
- 🚫 Unpurchased Courses → At least one course remains unsold, signaling potential for promotion.
- 📈 Learner Stats → 5 total learners, spread across 4 categories
